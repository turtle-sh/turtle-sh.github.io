turtle

Simple, fun command line

Turtle is a "shell" for javascript scripts. It provides some structure to make writing command-line scripts fun.

For instance, here's a date script, located at github.com/turtle-sh/date:

name: "date",

fn: function() {
  this.stdout.write( new Date() );
  this.exit();
}
$ date
> Mon Oct 07 2013 20:02:02 GMT-0400 (EDT)

Scripts can be requirejs'd in using the "install" script:

$ install turtle-sh/date
> installing "date" from https://github.com/turtle-sh/date
> date installed

The index.js file of a script acts as the equivalent of a package.json (Node.js) or manifest.json (Chrome Apps) file; its properties include some properties useful to command-line scripts:

tagLine

name: "date",
tagLine: "display date and time"

$ whatis date
> date  - display date and time

version

name: "date",
version: "0.1.0"

$ date -v
> v0.1.0
